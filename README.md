# 专利标记助手

一款专为专利撰写/审查人员设计的 Word 文档处理工具，支持附图标记自动标注、文本清洗与错别字检查。

---

## 主要功能

### 📌 标记提取与标注
- **自动提取附图标记**：从"附图说明"段落扫描并提取附图编号与部件名称（如 `1-齿圈，2-夹指`）
- **一键标注**：权利要求书加带括号编号（`齿圈（1）`），具体实施方式加无括号编号（`齿圈1`）
- **局部标注**：可单独标注权利要求书或具体实施方式
- **一键清洗**：将所有附图编号从正文中删除，还原为纯文字
- **格式安全**：底层操作 `<w:t>` XML 节点，**完整保留 MathType / AXmath 公式、图形等对象**

### 🧹 文本清洗
- **删除"所述"**：勾选章节后一键删除，可自由选择作用范围
- **标点检查**：半角→全角、全角→半角（可选），自动修正连续重复标点（如 `,,` → `，`）
- **孤立附图标记检测**：扫描附图说明中有而具体实施方式中未出现的标记编号

### 🔍 错别字 / 重复字词检查
- **内置专利词库**：覆盖权利要求书、实施例、结构件等常见错别字，可在界面中直接编辑、删除（含内置条目）
- **用户自定义词库**：支持添加、修改、导入（JSON/CSV）、导出，换机迁移方便
- **词库搜索**：支持关键字实时过滤，快速定位条目
- **重复字词检测**：自动识别 `所述所述`、`AA`、`ABCABC` 等连续重复词
- **重复字词忽略词库**：将常见"合理重复"添加后不再报告，支持搜索与导入导出
- **离线 NLP 增强**：可选安装 pycorrector 实现更深度的中文错字检测（软件内有零基础安装教程）
- **结果交互**：可编辑建议修改列，点"忽略"移除单条结果，一键应用所有修改

### 🖥️ 界面
- 浅色 / 深色主题一键切换
- 文档章节预览
- 操作历史记录
- 高 DPI 适配

## 快速上手

### 使用打包版（推荐，无需安装 Python）

直接运行 `dist/专利标记助手.exe`，无需任何额外安装。

### 从源码运行

```bash
# 安装依赖（需要 Python 3.10+）
pip install PyQt6 python-docx lxml

# 启动
python main.py
```

## 使用流程

**第一步：打开文档**
点击"📂 选择文件"或将 `.docx` 文件拖入窗口，软件自动识别权利要求书、具体实施方式等章节。

**第二步：确认标记字典**
软件自动提取附图标记显示在编辑框，如有遗漏可手动修改，改完点"🔄 重新确认标记"。

**第三步：执行操作**
- 标注页：点"⚡ 一键标注"，或分别点"仅标注权利要求书"/"仅标注具体实施方式"
- 清洗页：按需执行删除所述、标点检查、孤立标记检测、错别字 / 重复字词检查
- 应用修改后到标注页点"💾 文件生成"

**第四步：生成文件**
点"💾 文件生成"后，在原文件同目录下生成 `xxx_已标注.docx` / `xxx_已清洗.docx`。

## 项目结构

```
mark123/
├── main.py               # 程序入口
├── main_window.py        # 主窗口 UI 与交互逻辑
├── doc_parser.py         # docx 五书分段解析
├── annotator.py          # XML 级安全标注引擎
├── mark_extractor.py     # 附图标记提取
├── cleaner.py            # 文本清洗功能（删所述、标点、错别字、重复词）
├── typo_wordbank.py      # 内置专利错别字词库
├── wordbank_dialog.py    # 词库编辑器对话框
├── dup_ignore_dialog.py  # 重复字词忽略词库对话框
├── config_manager.py     # 配置与用户数据持久化
├── styles.py             # 深浅主题 QSS 样式
├── build.py              # PyInstaller 打包脚本
└── dist/
    └── 专利标记助手.exe  # 打包后的独立可执行文件
```

---

## 用户数据位置

软件将用户自定义配置保存在：

```
C:\Users\<你的用户名>\AppData\Local\专利标记助手\
├── user_wordbank.json          # 自定义错别字词库
├── disabled_builtin_wordbank.json  # 已禁用的内置词库条目
└── dup_ignore_wordbank.json    # 重复字词忽略词库
```
