# 专利附图标记助手 (Patent Marker)

专利附图标记助手是一款使用 PyQt6 和 python-docx 构建的桌面应用程序。主要用于辅助专利撰写和审查人员处理和分析专利申请文件（特别是 `.docx` 格式）中的附图标记和说明书内容。

## 主要特性 ✨

- **格式安全底层标注**：深入通过 `python-docx` 操作底层 `<w:t>` XML 标签执行批量标注与替换，**确保绝对保留 MathType、AXmath 公式对象和原生复杂文字格式**。
- **一键清洗功能**：除了一键增加附图标记，也支持根据标记字典一键**删除所有标记**（例如将 `齿圈（1）` 还原为 `齿圈`）。
- **双主题支持**：内置现代化的浅色(Light)与深色(Dark)玻璃态界面主题，提供直观的一键切换机制。
- **自动附图标记抽取**：一键提取说明书或段落中的所有附图标记，支持手动编辑并映射为正则表达式字典。
- **高DPI智能适配**：自动适配高分辨率屏幕并支持拖拽 Word 文档快速上传解析。
- **自动标点修复**：在全文范围内提供可选的标点清洗功能，自动修复由于合并或格式问题引起的 `,,` 或 `。。` 等冗余连续标点。

## 技术栈 🛠️

- **Python**: >= 3.14
- **GUI 框架**: [PyQt6](https://pypi.org/project/PyQt6/)
- **文档处理**: [python-docx](https://pypi.org/project/python-docx/)
- **依赖管理**: [uv](https://github.com/astral-sh/uv)
- **程序打包**: [PyInstaller](https://pyinstaller.org/)

这款“专利附图标记助手”桌面软件的操作流程非常直观，主要包含以下几个核心步骤：

**第一步：导入文档**
* 运行程序后，你可以点击主界面的“📂 点击选择 docx 文件，或将文件拖入此区域”按钮来选择文件，或者直接将标准的 `.docx` 格式的专利文档拖拽到软件窗口中完成导入。
* 文档加载后，软件会自动进行解析，智能识别出文档内的“权利要求书”、“具体实施方式”等五书章节。

**第二步：提取与核对附图标记**
* 导入成功后，软件会自动在文档中（通常是“附图说明”段落）扫描并提取附图标记及其对应的部件名称。
* 提取出的标记内容会显示在“📌 标记提取与标注”标签页的文本框内，格式形如 `1-齿圈，2-夹指`。
* 如果你发现自动提取的内容有遗漏或格式不准，可以直接在该文本框中进行手动修改或补充输入。
* 手动修改完成后，需点击下方的“🔄 重新提取标记”按钮，让软件重新读取并应用你编辑后的标记字典。

**第三步：执行标注或清洗操作**
核对完标记无误后，你可以在界面中部的操作区选择以下功能：
* **一键标注**：点击“⚡ 一键标注”按钮，软件会自动为“权利要求书”中的名称加上带括号的编号（例如“齿圈（1）”），同时为“具体实施方式”加上无括号的编号（例如“齿圈1”）。
* **局部标注**：如果你只想处理特定的某个部分，可以分别点击“📋 仅标注权利要求书”或“📝 仅标注具体实施方式”进行单独处理。
* **一键清洗**：若需要彻底删除正文中所有的附图编号并将文本还原，可以点击“🧹 删除所有标记”按钮进行全局清洗。

**第四步：获取处理后的文件**
* 操作结束后，软件会在原文档所在的同一文件夹下自动生成一个新的 `.docx` 文件，并根据你的操作在文件名末尾添加 `_已标注` 或 `_已清洗` 的后缀。
* 软件界面会弹出提示框并展示具体的段落替换数量信息，你可以点击弹窗中的“是”直接打开输出文件所在的文件夹。

**其他辅助与设置功能：**
* **文档预览**：你可以随时切换至“📄 文档内容预览”标签页，通过点击左侧的目录章节，在右侧面板快速预览该章节的正文内容。
* **自动标点修复**：界面顶部提供了一个“自动修正连续的标点”复选框，勾选此项后，软件在处理文档时会自动修复全文中因为格式或合并问题引起的冗余标点（如“。。”或“,,”）。
* **切换主题**：点击顶部的“🌓 切换主题”按钮，可以在内置的深色与浅色现代化界面之间自由切换。
