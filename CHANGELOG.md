## [4.0.0] · 2026-06-26 — 三区布局重构 + 内联富显示

里程碑版本：界面从「顶部标题栏 + 四大功能 Tab」整体重构为**三区布局**，专利内容成为常驻主舞台并支持**结构化实时编辑**；新增**图片 / 公式内联富显示**与**错别字 / 重复字 / 权项内联高亮**，检查结果与原文一屏对照。

### ✨ 新增

- **三区布局**：左上 **1框**（常驻内容主舞台）+ 左下 **2框**（当前子功能面板）+ 右侧 **3列 / 4列**（模块 → 子功能两列导航）。
- **1框 六个标签页**：权利要求书 / 说明书 / 说明书附图 / 说明书摘要 / **操作日志** / **操作历史**——正文与日志、历史统一汇总到一处。
- **结构化实时编辑**：1框 专利正文可就地编辑，点「✓ 确认修改」手动回写内存；**纯文本段可改、含图片 / 公式段只读**，且**禁止增删段落**，保证段落映射稳定、格式安全。
- **内联富显示**（新增 `ui/render/`）：含附图 / 公式的标签页按文档顺序内联渲染——图片（PNG / JPEG）直接显示，MathType / AxMath 公式（WMF / EMF 预览）经 Windows GDI 转图显示，**每段仍是一个 block、不增行**，行＝段模型不变；窗口缩放时按宽防抖重排。
- **内联高亮**：错别字 / 重复字 / 权项检查后，所有问题在 1框 对应位置**标黄**；单击结果表「修改前」/「说明」跳转到该处并**标红**强调，结果与原文一屏对照。
- **全文替换**（清洗模块）：对全文做格式安全的文本替换，保留公式 / 图片。
- **文件名跑马灯**：右侧文件按钮在长文件名时重绘式滚动展示完整名称，按钮宽度不被撑大。

### 🔧 变更

- 界面从 **4-Tab** 重构为**三区**：去掉顶部标题栏，内容区做主舞台（约占左侧 2/3 高度）。
- **错别字** 与 **重复字** 拆为两个独立模块（3列），共用 2框 同一结果表（标题随当前检查类型动态切换，计数并入标题）。
- 文件操作（上传 / 生成 / 打开目录 / 设置）整体移到右侧列底部。
- 三处可拖拽分隔条：**1框 ↔ 2框**、**左区 ↔ 右区**、**3列 ↔ 4列**；2框 可向下拖动收起。
- 结果表精简：权项检查表去「上下文」列、错别字 / 重复字表去「原文片段」列——原文已在 1框 内联标黄，不再重复占列。
- 错别字 / 重复字结果表新增「修改」列，可逐条应用单条修正。
- 目录分包：`core/`（纯逻辑）/ `config/` / `ui/`（含 `ui/dialogs/`）/ `infra/` / `tests/`，`main.py`、`version.py` 留根。

### 🗑️ 移除

- 「术语不一致检查」功能（误报噪音过大）；保留更实用的「不确定用语检查」。

### ✅ 兼容性

- 底层标注 / 清洗 / 检查引擎与 3.x 一致：仍只改 `<w:t>` 文本节点，公式 / 图形 / 样式在所有自动化操作后完整保留。
- 用户词库（错别字 / 重复字忽略 / 不确定用语）与界面设置的存储位置不变，从 3.x 升级无需迁移数据。
